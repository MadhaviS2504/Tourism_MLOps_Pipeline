# Tourism Wellness Package Prediction

This Gradio application predicts whether a customer will purchase the Wellness Tourism Package using a pre-trained machine learning model.

## Features
- Input customer demographics and travel preferences
- Get real-time predictions with confidence scores
- View input data used for prediction

## Model
The model is hosted on [Hugging Face Hub](https://huggingface.co/MadhaviSura/tourism-prediction-pipeline)
